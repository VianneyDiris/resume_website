Mastermind

C'est le jeu du mastermind en mode console.

Le projet contient 2 jeux fait en mode console. Le premier est un jeu de +/-, le but est de deviner le nombre avec comme indice que le nombre est plus grand ou plus petit. L'autre jeu est le mastermind dont le but est d'aussi de trouver le nombre mais avec des indices plus difficile.

Ce projet avait pour but de nous initier à la programmation java ainsi que de mettre en place un algorithme.

  • Date: Août 2017
  • Client: Openclassrooms
  • Categorie: Jeux

Escalade

Un site web sur le thème de l'escalade.

Le site web est fait pour parler d'escalade. On doit pouvoir voir la liste des sites d'escalade appelé Spot. Les utilisateurs doivent pouvoir s'authentifier et se partager des topos.

Ce projet avait pour but de nous apprendre à faire un site responsive ainsi qu'à appréhender les frameworks Struts pour le MVC et Spring IOC.

  • Date: Mars 2018

  • Categorie: Site web

Bibliothèque

Une application web pour une bibliothèque.

L'application a pour but de louer des livres comme dans une vrai bibliothèque. Au niveau des fonctionnalités:

  • Gestion du profil utilisateur
  • Fonction recherche par auteur/genre/titre
  • Relance par mail pour les livres non rendu

Ce projet avait l'ambition de nous apprendre comment communique 2 applications avec SOAP. Il m'a aussi permis de comprendre l'intérêt des batchs.

  • Date: Juillet 2018
  • Client: Openclassrooms
  • Categorie: Application web

Communautaire

Une application pour mettre en relation producteurs et clients.

Pour ce dernier projet du parcours, on nous a donné carte blanche, il faut juste que notre projet possède un aspect social/communautaire. Pour avoir un rapport plus professionnel, j'ai travaillé avec 2 autres étudiants. On a opté sur un application qui mettrait en relation les producteurs et les clients. On s'est rajouté comme difficulté qu'un client ne doit avoir accès qu'aux producteurs proche de chez lui.

Nous avons découpé notre application en microservice, pour mieux se partager les tâches. Je fûs en charge de la partie "gestion des produits et des stocks" ainsi que des adresses. J'ai développé aussi l'interface graphique concernant la gestion des produits au niveau des producteurs.

Ce projet est le plus complet que j'ai réalisé. Il comprend le travail d'équipe, l'utilisation de microservice Spring Boot REST et l'emploi de thymeleaf.

  • Date: Octobre 2018
  • Client: Openclassrooms
  • Categorie: Application web
